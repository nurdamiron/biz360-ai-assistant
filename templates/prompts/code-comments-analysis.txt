# Инструкции:
1. Проанализируйте предоставленные комментарии к коду и создайте структурированный анализ.
2. Для каждого комментария определите:
   - Тип комментария (вопрос, предложение улучшения, указание на ошибку, похвала, уточнение и т.д.)
   - Серьезность (критическая, важная, незначительная, информационная)
   - Требуется ли действие в ответ на комментарий
   - Суть комментария или предложения

3. В целом для всех комментариев:
   - Выделите повторяющиеся темы или проблемы
   - Определите наиболее критичные замечания
   - Сформулируйте общие рекомендации

4. Верните результат в JSON формате:
```json
{
  "commentAnalysis": [
    {
      "commentId": "<индекс или id комментария>",
      "type": "<тип комментария>",
      "severity": "<серьезность>",
      "requiresAction": true|false,
      "summary": "<краткое резюме>",
      "suggestedChanges": "<предлагаемые изменения, если есть>"
    },
    // другие комментарии...
  ],
  "overallAnalysis": {
    "mainThemes": ["<тема1>", "<тема2>", ...],
    "criticalIssues": ["<проблема1>", "<проблема2>", ...],
    "recommendations": ["<рекомендация1>", "<рекомендация2>", ...]
  },
  "summary": "<общее резюме всех комментариев в одном-двух предложениях>"
}
```

Верните ответ только в формате JSON, без дополнительных комментариев.
