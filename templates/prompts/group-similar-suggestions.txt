# Предложения и отзывы
{{#each suggestions}}

- Тип: {{this.type}}

# Инструкции:
1. Проанализируйте все предложения и отзывы и определите семантически схожие группы.
2. Объедините похожие предложения, даже если они сформулированы по-разному.
3. Для каждой группы:
   - Создайте заголовок, который отражает суть всех предложений в группе
   - Укажите все предложения, которые входят в эту группу
   - Укажите общее количество предложений в группе
   - Перечислите ID обратной связи, из которой взяты предложения
   - Определите общий тип предложений в группе (suggestion, issue и т.д.)

4. Верните результат в формате JSON массива:
```json
[
  {
    "text": "<заголовок группы>",
    "items": ["<предложение1>", "<предложение2>", ...],
    "count": <количество предложений>,
    "feedbackIds": ["<id1>", "<id2>", ...],
    "type": "<основной тип предложений>"
  },
  {
    "text": "<заголовок группы>",
    "items": ["<предложение1>", "<предложение2>", ...],
    "count": <количество предложений>,
    "feedbackIds": ["<id1>", "<id2>", ...],
    "type": "<основной тип предложений>"
  },
  // другие группы...
]
```

5. Правила группировки:
   - Группируйте предложения, если они касаются одной и той же функции или проблемы
   - Группируйте предложения, если их реализация была бы очень похожей
   - Не группируйте предложения только потому, что они относятся к одной категории (например, "UI")
   - Предложения, которые уникальны и не имеют похожих, могут образовывать группы из одного элемента

Верните ответ только в формате JSON массива, без дополнительных комментариев.
